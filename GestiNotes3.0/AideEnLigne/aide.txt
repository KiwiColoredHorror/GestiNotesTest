Bienvenue dans GestiNotes

Pour tester avec un jeu de huit eleves:
Aller dans Gestionnaire-Initialiser

Pour commencer avec vos classes:
Aller dans Eleve-Nouveau et ajouter les eleves l'un apres l'autre
(ils seront par groupe de 10)

Pour modifier les notes d'un eleve :
(requiert code permanent de celui-ci)
Aller dans Eleve-Modifier
Cliquer "Modifier un eleve" 
Entrer le code permanent de l'eleve
Apres avoir modifie les notes, 
Cliquer "Modifier un eleve" pour confirmer

Pour modifier les notes de plusieurs eleves:
Aller dans Gestionnaire-Modifier
Choisir le groupe puis l'eleve dans les listes deroulante

Pour lister les eleves d'un groupe:
Aller dans Gestionnaire-Lister
Selectionner le groupe voulu dans la liste deroulante
